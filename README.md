## 💡 产品想法：PodMetrics

### 核心问题
播客行业最大的商业化痛点：品牌方不敢投广告，因为看不到ROI。

根据2025年行业报告：
- 中文播客有2.2亿听众
- 但广告市场仅33亿元，远低于短视频
- 核心原因：广告效果无法量化

### 解决方案
一个AI驱动的播客广告效果追踪器：

**功能流程：**
1. 📤 播客主上传音频 + 标注广告时间段
2. 🤖 AI分析：
   - 听众在广告段的留存率（完播率）
   - 广告话术的情感倾向分析
   - 与同类播客的对比数据
3. 📊 生成专业的"广告效果报告"（可视化PDF）

### 目标用户：腰部和尾部播客创作者
- 月播放量：1万-10万次
- 典型困境：内容质量不错，但找不到广告主
- 核心需求：能证明广告效果的数据报告

### 技术栈
- 音频转文字：OpenAI Whisper API
- 数据分析：GPT-4
- 前端：Next.js + Chart.js
- 开发方式：Cursor + Claude AI辅助开发
